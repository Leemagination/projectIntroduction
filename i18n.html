# 国际化

# 项目背景

WMS系统和开放平台系统是面向社会化的系统,需提供多语言功能以便于海外仓库员工和客户的使用

# 使用技术

Angular + ngx-translate
Vue + vue-i18n

# 项目流程

  • 因仓储系统更新迭代快,新的模块会有新的名词出现。国际化功能需搭建一个配置页面来保存翻译的数据， 前端从后端获取翻译的数据，通过开源的国际化功能库进行翻译。
  • 当页面遇到未经翻译的词语时会自动向后端发送请求，后端通过翻译软件的API将数据翻译并存入数据库，当页面重新加载翻译数据便能正常翻译。 若自动翻译不准确并不适用于系统时只需运营人员去配置页面手工修改翻译文本即可。
